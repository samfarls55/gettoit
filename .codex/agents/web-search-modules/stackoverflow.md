# Stack Overflow Module

Use for technical Q&A, API usage examples, and common implementation mistakes.

## Priority Sources

- Stack Overflow
- Relevant Stack Exchange sites
- Official forums and language/framework discussion boards

## Query Strategy

- Search exact error text plus the framework or API name.
- Prefer answers with recent edits, runnable examples, and comments confirming current behavior.
- Cross-check accepted answers against official docs when the answer is old.
- Watch for version drift and deprecated APIs.
