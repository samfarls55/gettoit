# Chinese Tech Module

Use for Chinese-language technical communities, domestic frameworks, and cases where English-language results are sparse.

## Priority Sources

- CSDN
- Juejin
- SegmentFault
- Zhihu
- Cnblogs
- OSChina
- V2EX
- Tencent Cloud developer articles
- Alibaba Cloud developer articles

## Query Strategy

- Generate searches in both English and Chinese when the topic may have regional vocabulary.
- Include Chinese terms for errors, fixes, and examples.
- Cross-check Chinese community fixes against official docs or source code when possible.
- Treat copied blog content cautiously; prefer posts with original examples, dates, and environment details.
