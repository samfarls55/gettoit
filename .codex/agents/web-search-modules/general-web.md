# General Web Module

Use for best practices, comparative research, product research, and current ecosystem context.

## Priority Sources

- Official documentation and changelogs
- Maintainer blogs or release notes
- Production code examples
- Reddit and Hacker News discussions
- Dev.to, Medium, and technical blogs
- Discord, forum, or X/Twitter announcements when official sources point there

## Query Strategy

- Start with official documentation or changelogs.
- Search for current best practices and deprecated approaches.
- Compare multiple implementations before recommending one.
- Capture dates, version numbers, and known tradeoffs.
- Separate official guidance from community workarounds.
