# Academic Papers Module

Use for paper search, academic research, and algorithm or theory background.

## Priority Sources

- Google Scholar
- arXiv
- Hugging Face Papers
- bioRxiv
- Semantic Scholar
- ACM Digital Library
- IEEE Xplore
- Publisher or university-hosted PDFs when legally available

## Query Strategy

- Search exact paper titles, author names, DOI numbers, institutions, and publication years.
- Use year ranges to separate seminal work from recent follow-up work.
- Look for related papers, citation trails, and survey papers.
- Prefer primary papers and official publisher/preprint pages over secondary summaries.
- Note publication venue, citation context, and whether a result is a preprint.
