# GitHub Debug Module

Use for debugging, version-specific bugs, regressions, and error-message investigation.

## Priority Sources

- GitHub Issues, both open and closed
- GitHub Pull Requests
- Release notes and changelogs
- Source code around the failing API or behavior

## Query Strategy

- Search exact error messages in quotes.
- Include package, framework, and version names.
- Search both the failure and likely workaround language.
- Read closed issues for resolution patterns.
- Check if a fix landed in a newer version or only on an unreleased branch.
