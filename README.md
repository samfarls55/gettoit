# GetToIt

Group decision-paralysis killer. Food vertical 0.1.0.

## Repo Map

- `web/` - Next.js web fallback.
- `mobile/` - Active Expo managed React Native iOS client.
- `ios/` - Retired SwiftUI iOS client kept as legacy reference.
- `supabase/` - migrations, Edge Functions, and Supabase local config.
- `design-system/` - tokens, components, surfaces, motion, and accessibility specs.
- `gti-vault/` - durable product, design, engineering, and issue context.
- `docs/agents/` - agent-facing contracts and verification notes.

Start with `AGENTS.md` for repo instructions. Use `docs/agents/verification.md` for local check selection.
